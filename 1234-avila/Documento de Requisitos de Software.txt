#Documento de Requisitos de Software

### 2.1. Requisitos Funcionales

-Busqueda de contenido
-Interacción entre Usuarios
-Personalización de Perfiles
-Autenticación y gestión de usuarios
-Reproducción de música
-Gestión de listas de reproducción
-Descurbimiento y búsqueda de Música
-Recomencaciones personalizadas
-Interacción Social
-Historial de Reproducción
-Configuración y Preferencias
-Integración con otras Aplicaciones
-Gestión de Planes de Suscripción
-Publicidad y Monetizacion (cuentas gratuitas)
-Biblioteca y Colección de Música

### 2.2 Requisitos De Diseño
-Colores del software y su logo
-Los colores del software son el celeste y negro. 

### 2.3. Requisitos No Funcionales.
-Seguridad: inicio de sesión de cada usuario.
-Usuarios: 1000000
